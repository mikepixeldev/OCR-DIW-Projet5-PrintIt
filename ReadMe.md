# Print it JS Project 5 OPC

## Description

Ce projet est un fork du "Print it JS Project 5" d'OpenClassrooms, destiné à mettre en pratique les compétences en développement web, notamment en JavaScript. Il consiste en la création d'un carrousel d'images interactif pour le site web d'une entreprise fictive, Print it, offrant une expérience utilisateur dynamique et engageante.

## Fonctionnalités

- Carrousel d'images interactif avec navigation par flèches.
- Indicateurs de navigation (bullet points) pour chaque image du carrousel.
- Fonction de défilement infini du carrousel.
- Design responsive pour une utilisation sur différents appareils.

## Technologies Utilisées

- HTML5
- CSS3
- JavaScript

## Structure du Projet

```
Print-it-JS-Project-5/
│
├── assets/             # Ressources utilisées dans le projet
│   ├── images/         # Images pour le carrousel et autres éléments du site
│   ├── script.js       # Script JavaScript pour le fonctionnement du carrousel
│   └── styles.css      # Feuille de style CSS pour la mise en page du site
│
└── index.html          # Page principale du site web
```

## Installation

Pour utiliser ou modifier ce projet :

1. Clonez le dépôt :
   ```
   git clone https://github.com/mikepixeldev/Print-it-JS-Project-5-OPC
   ```
2. Ouvrez `index.html` dans un navigateur web pour visualiser le projet.
